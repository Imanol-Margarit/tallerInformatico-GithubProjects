# TallerInformatico-GithubProjects
Historias de usuario:
·Como recepcionista, quiero registrar la entrada de un equipo con los datos del cliente para poder hacer seguimiento de la reparación.
·Como técnico, quiero registrar el diagnóstico del equipo para documentar el problema detectado.
·Como administrador, quiero gestionar proveedores y piezas para controlar el inventario.
·Como administrador, quiero generar presupuestos y enviarlos al cliente para su aprobación.
·Como técnico, quiero registrar el avance de la reparación para mantener el estado actualizado.
·Como administrador, quiero gestionar garantías e incidencias post-reparación.
·Como usuario del sistema, quiero ver la planificación de tareas para organizar mejor el trabajo.
